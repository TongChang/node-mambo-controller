# node-mambo-controller

Node.js on Raspberry Pi から Parrot MAMBO Fly の操作を試行するリポジトリです。

## 環境

- Raspbian version.

```$ cat /etc/debian_version```
```> 8.0```

- Node.js version.

```$ node -v```
```> v6.11.0```

## やったこと

```$ sudo apt-get install bluetooth bluez libbluetooth-dev libudev-dev```
```$ npm install --save noble```
